Commanade de radiateur via fil pilote avec arduino

based on  https://www.instructables.com/E-Paper-Display-Partial-Refresh-With-Arduino/

I wired the display to the Arduino Nano as follows:

Display <-> Arduino

BUSY <-> D7

RST <-> D8

DC <-> D9

CS <-> D10

DIN <-> D11

CLK <-> D13

VCC <-> 3.3V

GND <-> GND

RTC wiring
